What was the hardest problem you encountered that you were able to fix?

The hardest problem I encountered was being able to efficiently convert numbers into words for the cheque writer.

Is your solution elegant/optimal/efficient?

I managed to find an efficient solution, although it probably isn't optimal. The solution was to use arrays for the ones place, teens, tens place and hundreds, then make a function to convert up to 100. Then I made a loop that broke the number up into chunks of 3, then converted each chunk to words using the function and added the place value (hundreds, thousands, millions, etc.) onto the end, and combined it all in a string.
